# 🌱 Plant Pal Tracker

Plant Pal Tracker is a web application designed to help plant owners organize and manage their plant care routines. The application allows users to track individual plants, monitor watering schedules, log care activities, and maintain plant health records in one centralized platform.

Whether you're a beginner plant parent or an experienced gardener, Plant Pal Tracker helps ensure your plants receive consistent care through an intuitive and user-friendly interface.

## ✨ Features

- 🌿 Add and manage plant profiles
- 💧 Track watering schedules and watering history
- 📅 Schedule and monitor plant care tasks
- 📝 Record notes and plant health updates
- 📊 View plant information and maintenance history
- 📱 Responsive design for desktop and mobile devices

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- React

## 🎯 Project Purpose

Plant Pal Tracker was developed to provide plant owners with a simple and organized way to manage plant care. This project demonstrates full-stack web development concepts, including database management, CRUD operations, responsive design, and user-focused application development.
